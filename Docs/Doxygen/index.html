Defender:SA - Overview
Authors
@JoshSchwark (Joshua Schwark) and @BrentTB (Brent Butkow)
  • Defender:SA is a computer based game that is based on the 1980's game Defender
  • It is a 2D arcade game that is made in c++ and uses SFML for rendereing purposes

Version History

v1.0

First Release:
This game is an unfinished project, based on the arcade game Defender
The current version of the game has the following features:

  • There is a splash screen that correctly tells the user what keys to press
  • The User spaceship can fly around the map, and can fire lasers
  • Lander enemy’s spawn around the player, and shoot at the player
  • Bomber enemy’s spawn and fly around the map, leaving mines
  • The laser can shoot the enemy’s, and can be destroyed by the enemy’s lasers, bombs, or the enemies themselves
  • The game ends when the Player is destroyed
  • The player has three shields, which can be activated to protect it from collisions
  • The screen moves around the player when the player moves towards the edges

v2.0

Second Release:
This game is an unfinished project, based on the arcade game Defender
The current version of the game has the following features:

  • There is a splash screen that correctly tells the user what keys to press
  • Good Graphics are implemented
  • The game can be restarted when the player dies by pressing enter
  • A score system is implemented
  • The score is increased when landers and bombers are destroyed
  • The highscore from across all games is shown, and updates if the user beats the highscore.
  • Highscores are retained across the running of the executable
  • Humanoids spawn randomly around the screen. Landers try to kidnap them, and the player must defeat the lander and catch the human before it dies
  • Humanoids can die by getting shot by the Player, when the lander abducting it reaches the top of the screen, or by falling to the bottom of the screen
  • When Humanoids die, the player's score is decreased
  • The number of shields the user has can be seen on the game screen
  • The landscape is depicted and the game scrolls as the player’s ship flies towards either edge of the screen
  • When all enemies have been defeated, the player levels up, and more enemys can spawn, and their spawn rate increases
  • All mines disappear when the player levels up
  • Asteroids spawn randomly and fall in groups (asteroid showers) from the top of the screen, and can kill the player and enemies
  • The player has a limited amount of fuel, which can be replenished with fueltanks at the bottom of the screen, which appear and disappear randomly
  • The amount of fuel the player has can be seen on the game screen
  • When player's run out of fuel, they fall out of the sky and die

v3.0

Final Release:
The final version of the Defender:SA game
The current version of the game has the following features:

  • There is a minimap that displays where the player is on the screen
  • The number of shields resets on level-up
  • There is a Level Up animation when the player levels up
  • The player moves to the opposite side of the screen when moving from side to side
  • Score achieved is displayed on the end screen
  • An information screen has been added when I is pressed
